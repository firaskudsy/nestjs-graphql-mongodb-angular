# NgxNestMongoose

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 8.0.1.

## Development server

Run `ng serve` for a dev server. Navigate to `http://127.0.0.1:4200/`. The app will automatically reload if you change any of the source files.

## Build Server & Clinet dist

Run `npm run serve` to build the project. The build artifacts will be stored in the `dist/` directory.

## Start Server

Run `npm run start:prod` to start the server.

## Access the GraphQL Playground

Open the Browser on `http://127.0.0.1:4000/graphql` to start the server.
